# 12.02 Взаимодействия с досками: редактирование, архивирование, копирование и создание авто-действий

Публичная ссылка: https://app.manajet.org/FAQ/Public/Index/483?culture=ru-RU

Различные возможности взаимодействия с доской/столбиком/карточкой производятся через кнопку .
Кнопка взаимодействия предоставляет возможность редактировать, копировать, архивировать доску, а также добавлять автоматические действия:
При нажатии на кнопку “Редактировать” откроется окно:
Вкладка “Общие”
Название
Здесь можно редактировать название доски.
Показать завершенный статус
Галочка в этом пункте даёт возможность отслеживать процесс работы с карточками (завершённые они или нет).
Показать ID карточки
Предоставляет возможность отображать ID (идентификатор; внутренний, уникальный номер записи). С помощью ID легче определить ту или иную карточку в Потоке.
Финансовое Планирование
Позволяет прикреплять карточки к периодам финансового планирования.
Доступ
С помощью кнопок  и  можно предоставить доступ к доске должностям или подразделениям.
Вкладка “Статистики”
Эта вкладка позволяет прикреплять к доске статистики, чтобы при открытии доски сразу можно было увидеть необходимые показатели.
Например, в доске "Работа с клиентами" может быть прикреплена статистика выручки.
Здесь есть зелёная кнопка "Добавить" для создания новой статистики, а также кнопки:
"Все"
"Личные"
"Статистики подчинённых"
Это упрощает поиск статистики. Также присутствует окно поиска и кнопка для расширенного поиска .
Фильтры в расширенном поиске:
Окно поиска
Выбор компании, к которой привязана статистика
Рост (Да / Нет / Все)
Период ввода (Не выбран / Дни / Недели / Месяц / По кварталам / Год)
Ответственный сотрудник (выпадающий список)
Архив (Активные / В архиве / Все)
Тип статистики (можно выбрать из списка или ввести название вручную)
Не использовать / Процентный рост
Период просмотра (Не выбран / Дни / Недели / Месяц / По кварталам / Год)
Не использовать / Показать дату / Значения введены и дата напротив
Здесь можно выбрать нужную статистику, отметив её галочкой, и сохранить изменения.
Прикрепленные статистики будут отображаться на доске следующим образом:
Вкладка “Должностные инструкции”
Эта вкладка в окне редактирования доски "Поток" (должностные инструкции) позволяет прикреплять инструкции к доске.
Основные возможности:
Выбор инструкций – можно отметить нужные инструкции галочками.
Кнопки управления:
Добавить – создание новой инструкции.
Прикрепить инструкцию – прикрепление выбранных инструкций к доске.
Фильтр "Все / Не ознакомлен" – позволяет отфильтровать инструкции по статусу ознакомления.
Поле поиска – можно найти нужную инструкцию по названию.
Можно выбрать нужную должностную инструкцию (или же несколько) и прикрепить к доске.Прикрепленная инструкция будет отображаться следующим образом:
Чтобы открыть должностную инструкцию, нужно нажать на её название (отображается синим цветом):
Вкладка “История”
Вкладка "История" отображает полный журнал изменений в доске, фиксируя все действия пользователей.
Что можно увидеть:
Тип события (создание, изменение и другие).
Дата и время внесения изменений.
Имя пользователя, который совершил действие.
Описание изменений, например:
Перемещение карточки между списками.
Изменение статуса задачи (например, "Выполнено").
Архивация карточки.
И др.
Копирование карточек
Данная кнопка позволяет скопировать доску со всеми данными, находящимися в ней (этапы, карточки).
Авто-действия
Кнопка "Авто-действия" позволяет настроить автоматизацию некоторых процессов в "Потоке". С её помощью можно, например, автоматически назначать ответственного за карточку при её перемещении в определённый список.
Подробнее о настройке авто-действий можно узнать в отдельной статье в справке
Архив
Кнопка "Архив" позволяет отправить доску в архив, скрыв её из основного списка. Архивированные доски сохраняются и могут быть восстановлены при необходимости.
