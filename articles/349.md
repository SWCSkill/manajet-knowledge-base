# 10.08.5 Контроль средств. Панель "Подробно"

Публичная ссылка: https://app.manajet.org/FAQ/Public/Index/349?culture=ru-RU

Панель “Подробно” позволяет просмотреть данные по контролю средств в виде графиков и диаграмм. Чтобы открыть данную панель необходимо кликнуть  в верхней части страницы Контроль средств:
Появится 2 дополнительных графика и 2 диаграммы:
Статистика, отражающая общие суммы счетов Контроля средств за периоды.
Мультистатистика, отражающая отдельные значения каждого из счетов по периодам.
Диаграмма, отображающая значения сумм счетов за предыдущий период в процентном соотношении.
Диаграмма, отображающая значения сумм счетов за текущий период в процентном соотношении.

Контроль средств при работе с иностранными валютами
Таким образом выглядит вкладка “Контроль средств” при подключенной мультивалютности:
В нижней части экрана(Контрольная сумма) меняется вид отображения средств:
Состояние средств по каждой валюте, ранее добавленной в программу. В зависимости от количества валют число столбиков может увеличиваться.
Суммарное значение средств по всем валютам(отображается в валюте, которая обозначена “по умолчанию”).
Верхняя часть Контроля средств, где вносятся данные по счетам компании приобретает вид:
Появляются дополнительные строки ввода для каждой из валют, которая внесена в программу. Состояние каждой валюты вносится по отдельности в соответствующую строку.
