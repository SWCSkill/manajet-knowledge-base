Публичная ссылка: https://app.manajet.org/FAQ/Public/Index/29?culture=ru-RU

Для того чтобы на плазме добавить элемент, нужно на верхней панеле нажать на , и из появившегося списка выбрать нужный Вам элемент.
На плазме можно добавить следующие элементы:
- элемент который будет показывать время;
- добавление текста на плазму;
- добавление столбика, который в дальнейшем будет отображать данные;
- набор столбиков, столбчатая диаграмма, где каждый столбик будет обозначать отдельное данное(из статистики);
- график;
- добавление картинки/фотографии;
- построение таблицы;
- круговая диаграмма, которая показывает соотношения выбранных значений;
- направляющие формы.

9.   Бизнес-процессы.
Бизнес-процессы - специальный документ, содержащий определенный маршрут движения той или иной частицы(публики, сотрудника) по линиям организации. напротив каждого пункта направляющей формы отведено специальное поле, в котором расписывается сотрудник, который должен был выполнить указанное в этом пункте действие(в том случае, если он его выполнил). Форма считается завершенной, когда все пункты, указанные в ней, выполнены, о чем свидетельствуют инициалы напротив каждого из них.Опять таки для настройки выведения отчета по направляющим формам нужно настроить все надстройки на вкладке “Данные”:
