# 18.2 Расширение ManaJet к "1С Управление Торговлей" - настройка

Публичная ссылка: https://app.manajet.org/FAQ/Public/Index/170?culture=ru-RU

Если вы читаете эту инструкцию, значит необходимые расширения в 1С установлены, и мы можем заняться их настройкой. (Если это не так, то читайте  инструкцию «Манаджет УТ установка».)
Для начала нам нужно настроить подключение, для этого открываем раздел «Манаджет», выбираем пункт «открыть настройки»
В появившемся окне необходимо заполнить:
«Секретный ключ» - это API-ключ ManaJet. Создать его или скопировать можно в разделе ManaJet: «Настройки / API ключи»
Пароль – пароль от учётной записи ManaJet, которую вы планируете использовать для синхронизации
Пользователь – логин пользователя от учётной записи ManaJet, которую вы планируете использовать для синхронизации.
Адрес подключения - https://api.manajet.org/api
Компания – наименовании компании (аккаунта) в ManaJet
Для того чтобы сохранить данные нажимаем «Записать», после этого 1С попробует установить подключение и если все данные введены корректно, то данные будут сохранены, если нет то будет выдано сообщение об ошибке.
Примечание:Чтобы расширение 1С можно было подключить к API ManaJet, на сервере должна быть действующая лицензия на расширение.
Примечание:
Чтобы расширение 1С можно было подключить к API ManaJet, на сервере должна быть действующая лицензия на расширение.
Затем нужно настроить расписание, по которому 1С будет выгружать необходимые данные. Для этого заходим в раздел «Настройки отправки сообщений» и нажимаем на синюю ссылку:
В открывшемся окне производим настройку расписания.
Следующим шагом нам понадобится настроить соответствие статей учёта ManaJet и 1С. Сделать это можно в окне «Настройка дополнительных реквизитов», так же перейдя по синей ссылке «Настройка и привязка статей к операциям».
На этом первоначальная настройка подключения 1С к Манаджет завершена.
