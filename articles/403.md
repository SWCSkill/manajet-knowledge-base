# 21.1 Интеграция  с Shopify

Публичная ссылка: https://app.manajet.org/FAQ/Public/Index/403?culture=ru-RU

В ManaJet существует интеграция с Shopify. C её помощью можно синхронизировать информацию о товарах и заказах в ManaJet: заказы перенаправляются в счета клиентов в блоке Финансового Планирования , а товар синхронизируется с ассортиментом в разделе CRM .
Это обеспечивает централизованное управление данными и упрощает бизнес-процессы. Интеграция с Shopify позволяет автоматизировать обновление информации о товарах и заказах, минимизируя вероятность ошибок и сокращая время, необходимое для ручного ввода данных. Также это способствует более эффективному взаимодействию с клиентами, поскольку информация о заказах и товарах всегда актуальна и доступна в одной системе.
Настроить интеграцию с Shopify можно в Общих настройках , кликом на иконку  :
В открывшемся окне будет представлена информация о любых ранее установленных интеграциях.:
Чтобы подключить новую интеграцию, кликните на иконку. Появится форма для ввода данных новой интеграции:
Заполняем все необходимые данные.
Магазин Shopify — введите название вашего магазина на платформе Shopify.
Секретный ключ Shopify Webhook – введите соответствующие данные, полученные из аккаунта Shopify.
Ключ токена API Shopify — введите соответствующие данные, полученные из аккаунта Shopify.
Папки — можно выбрать заранее созданную папку ассортимента из CRM “Ассортимент”,  в которую будут добавляться карточки товара.
Доход – из выпадающего списка выбирается вид дохода, который будет использоваться при создании счёта.
Расходы — из выпадающего списка выбирается вид расхода, который будет использоваться при создании счёта.
Способ оплаты — указывается способ оплаты, который будет использоваться при создании счёта.
Товарная карта для доставки — если есть доставка через Shopify, выберите соответствующую карточку товара.
