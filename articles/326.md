# 10.03.4 Фонды. Обмен валют между фондами

Публичная ссылка: https://app.manajet.org/FAQ/Public/Index/326?culture=ru-RU

Обмен валют между фондами предназначен для внутреннего обмена денежных средств – из одной валюты, в другую. При этой операции не будет потерь от конвертации и общая сумма денег на каждом из задействованных фондов не поменяется.
