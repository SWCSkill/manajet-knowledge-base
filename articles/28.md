# 8.2 Создание/редактирование плазмы

Публичная ссылка: https://app.manajet.org/FAQ/Public/Index/28?culture=ru-RU

Для того чтобы создать новую плазму, необходимо нажать на кнопку "Создать плазму":
После этого появляется окно добавления плазмы с двумя вкладками -  "Общая информация" и "Персонал":
На вкладке “Общая информация” заполняем название и описание ( по желанию), если Вы хотите сразу после создания перейти к самой плазме, то для этого нужно поставить галочку, как показано на изображении выше.
На вкладке “Персонал” мы отмечаем тех у кого будет просмотр к данной плазме. Если Вы хотите чтобы каждый сотрудник смог просмотреть данную плазму нужно поставить отметку “Общедоступная”.
Чтобы было доступно редактирование плазмы необходимо сотруднику в его карточке поставить галочку возле права "Редактирование плазм".
