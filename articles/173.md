# 14.1 Дополнительные поля

Публичная ссылка: https://app.manajet.org/FAQ/Public/Index/173?culture=ru-RU

Дополнительные поля дают возможность сохранять в ManaJet типы данных, которых нет в программе изначально. Таким образом можно подстроить программу под свой бизнес.
С помощью дополнительных полей, например, в карточку сотрудника можно добавить его контакт в Skype  (изначально такого поля нет), дополнительный е-мейл или телефон, а также другие данные:
Дополнительные поля можно использовать в следующих разделах ManaJet:
CRM:
-Компании
-Контакты
-Счета поставщиков
-Счета клиентов
-Ассортимент
Финансовое Планирование:
-Счета поставщиков
-Счета клиентов
Сотрудники
ManaJet Поток

Обязательно для заполненияПри отметке этого пункта рядом с созданной колонкой появится значок . Он означает, что без заполнения этого поля программа не даст сохранить запись – ни при создании, ни при редактировании.
Показывать на основной вкладкеЭта отметка появляется только при создании дополнительных полей в разделе “Компании”, в CRM. Если её включить, то поле будет показано во вкладке “Основные” – т.е. на первой странице профиля компании. Если она не включена, то Дополнительное поле будет показано во вкладке “Дополнительно” (т.е. на второй странице).
