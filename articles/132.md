# 10.05.3 Счета поставщиков. Одобрение счетов

Публичная ссылка: https://app.manajet.org/FAQ/Public/Index/132?culture=ru-RU

Одобрение счетов поставщиков выполняется во вкладке “Счета поставщиков”. Чтобы войти в нее, нужно кликнуть на иконку , а затем перейти на вкладку “Счета поставщиков”.
Выбираем счет, который нужно одобрить. Делаем клик на иконке  панели счетов:
Откроется форма для одобрения:
ФП Период
Программа устанавливает по умолчанию текущий период ФП.
Сумма
По умолчанию программа предлагает одобрить полную сумму счета, но можно сделать и частичное одобрение.
Источник
Программа предложит фонд для одобрения счета, но можно выбрать другой.
Вид расхода
Программа предложит статью расхода по одобрению счета, но можно выбрать другую.
Затем нужно нажать “Ок”.
