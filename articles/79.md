# 3.7 Добавление группы сотрудников

Публичная ссылка: https://app.manajet.org/FAQ/Public/Index/79?culture=ru-RU

В программе есть возможность создавать группы  сотрудников вашей компании, например: по территориальному принципу, штатные сотрудники, аутсорс-персонал, по наличию рабочего места в офисе или дома, по степени квалификации, способ документального оформления (трудовой договор, ГПХ, совместительство) и тд. Для этого нужно перейти на вкладку ”Группы”:
Далее, нужно нажать на кнопку , после чего откроется окно создания группы:
После того, как Вы нажмете на кнопку для сохранения, программа нас вернет к нашему списку групп:
Если вернуться обратно на вкладку сотрудники, то все группы которые созданы будут находить  в левой части экрана:
Нажимая на группу, в списке сотрудников автоматически покажет только тех сотрудников, которые входят в эту группу.
Для того чтобы добавить сотрудника в ту или иную группу, необходимо зайти в анкету сотрудника и выставить там созданную группу.
Также, на вкладке “Группы” в правом верхнем углу помимо строки для поиска и кнопки для расширенного поиска есть кнопка для печати списка групп. На напечатанном экземпляре Вы увидите название групп, название группы в которую данная группа входит, количество человек.
Примечание:Не рекомендуем создавать группы сотрудников по их функциям: "учредители", "директора", "производство", "маркетинг" и тд. . Сотрудники уже разделены по данному принципу на постах в Структуре.И важное отличие состоит в том, что в Структуре одному и тому же человеку можно назначить должности из разных подразделений (неограниченное количество), а группу сотруднику можно назначить только одну. Поэтому группы сотрудников лучше использовать по другим общностям.
