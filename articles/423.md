# 24.56 Обновлённая функция "Отгрузка" в ManaJet!

Публичная ссылка: https://app.manajet.org/FAQ/Public/Index/423?culture=ru-RU

Встречайте свежие обновления функции “Отгрузка” в ManaJet!
Сделайте работу со счетами и ассортиментом более гибкой и эффективной! 🚀 Теперь в ManaJet есть несколько вариантов управления отгрузками:
💰 Отгрузка по выплаченной сумме счёта — этот традиционный метод позволяет проводить отгрузку товаров на сумму, равную фактически поступившим платежам по счету. Идеально подходит для строгого контроля финансовых потоков.
📦 Отгрузка по сумме счёта (представленной к оплате) — новая возможность, позволяющая отправлять товары на полную сумму счета, даже если платежи ещё не поступили и счёт имеет статус “не оплачен”. Это удобно при работе с проверенными клиентами и работой с отсрочкой платежа.
🔒 Отгрузка по одобренной сумме счёта — новая возможность, позволяющая отгружать товар на сумму, которая была одобрена клиенту в качестве кредитного лимита. Подходит для работы с постоянными партнёрами, имеющими договорные условия по кредитованию.
