# 12.0 "Поток" — обзор возможностей

Публичная ссылка: https://app.manajet.org/FAQ/Public/Index/320?culture=ru-RU

Поток – инструмент для простой и наглядной работы с  бизнес-процессами компании.
Его можно применять для любых процессов, где есть этапы выполнения, в самых разных сферах и сторонах деятельности компании: для обработки заявок с сайта (воронка продаж), при найме и увольнении сотрудников, для подготовки документов бухгалтерией, контроля рабочих процессов на строительных объектах и тд.
Для каждого бизнес-процесса в “Потоке” создаётся доска с колонками и карточками. Одна колонка – один этап процесса. Карточка представляет собой “частицу” – задачу, сделку, заказ, документ и т.д., – которую нужно обработать. По мере выполнения карточку перемещают по доске слева направо.
Таким образом, доска является удобным инструментом как для выполнения процесса, так и для показа его состояния: какие задачи сейчас в работе, на каком шаге, кто за какую задачу отвечает и тд.
Можно управлять правами доступа сотрудников к доскам – создавать доски как общедоступные, так и видимые только определённым сотрудникам, подразделениям и командам в компании.

Если добавить дополнительные колонки не в карточку, а сразу на доску процесса, то они автоматически появляются во всех карточках этой доски. И по дополнительным колонкам будет доступен поиск карточек:
В карточках “Потока” можно указывать метки – это ещё один удобный инструмент ManaJet. В “Потоке” метками обозначают различные типы “частиц” – выполняемых задач. Для IT-компаний, это, например, разработка новых функций, сообщения об ошибках, задачи по обслуживанию системы и тд. То, что в бизнес-процессе необходимо различать, но не хочется использовать для этого дополнительные колонки.

В Потоке есть три режима отображения карточек. Для выбора режима нужно кликнуть на одну из иконок  на верхней панели доски:
Карточки
По умолчанию карточки отображаются на доске именно в этом режиме – по колонкам. Одна колонка – один этап процесса:
Машина времени
В этом режиме карточки автоматически сгруппированы в зависимости от времени последних действий с ними: карточка добавлена, были изменения (переход с этапа на этап) или же задача была завершена.
Колонки по времени последних действий следующие: “Сегодня”, “Вчера”, “В течение трёх дней”, а также в течение одной и двух последних недель.
Карточки, с которыми не было действий в течение двух последних недель, в этом представлении не отображаются.
