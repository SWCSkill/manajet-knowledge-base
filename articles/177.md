# 5.10.2 Дополнительные поля

Публичная ссылка: https://app.manajet.org/FAQ/Public/Index/177?culture=ru-RU

Настройка и создание полей
В разделе CRM можно добавлять дополнительные поля, они же поля с информацией для заполнения в карточке компании. Также есть возможность выводить эти поля в основной раздел CRM для быстрого и удобного доступа к нужной информации по компаниям. Для добавления поля зайдите через шестерёнку в раздел “Дополнительные поля”:
В открывшемся меню нажмите кнопку  и выберите “Тип поля”, его “Название”, а также “Описание”.
Галочка “Показывать в списке”, означает что это поле можно будет выбрать для того, чтобы отображать в колонке в основном разделе CRM.
Галочка “Обязательно для заполнения” означает, что без заполнения этого поля создать компанию будет невозможно.
Галочка “Показывать на основной вкладке” означает, что это поле будет выводиться при создании карточки компании и еще это поле будет отображаться в самих карточках компании.

Дополнительно, давайте разберем Типы полей:
Адрес — здесь указывается адрес, который сверяется с яндекс-картами для того, чтобы адрес был введен правильно.
Да/нет — табличка, которую можно переключать внутри карточки. Похоже на отметку галочкой окошка
Дата — дата, которую можно выбрать из календаря.
Десятичное число — с помощью данного дополнительного поля можно добавить любое число от 0 до 10.
Компания — позволяет добавить любую карточку компании, которая заведена в CRM.
Контрольный список — с его помощью можно указывать несколько пунктов списка с возможностью “вычёркивания” по мере выполнения.
Персонал — список с Вашими сотрудниками.
Ссылка URL — позволяет добавить ссылку на какой-либо сайт, который заканчивается на .ru .com и т.д.
Счет клиенту — даёт возможность выбрать какой-либо созданный счет клиенту, например главный текущий
Счет поставщика — даёт возможность выбрать какой либо созданный счет поставщика, например главный текущий.
Текст — позволяет вписать любой текст, в том числе цифры, с ограничением в одну строку.
Текст многострочный — возможность вписать любой текст, в том числе цифры, в большом формате и без ограничений.
Файлы — позволяет добавлять любые файлы с вашего компьютера (картинки, документы Word и Exel и др.).
Целое число — даёт возможность вписать любое число.
