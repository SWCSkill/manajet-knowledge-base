# 15.1 Интеграция

Публичная ссылка: https://app.manajet.org/FAQ/Public/Index/128?culture=ru-RU

Подробное описание методов доступно по адресу http://api.adminsolution.org/
Для того чтобы воспользоваться данным функционалом, необходимо получить авторизационный ключ для подключения к программе. Он состоит из двух частей: название компании и секретного слова, которое можно создать в разделе Настройки => API Ключи. Пример ключа: compas:0580fa88-e889-4f88-9e98-fed67e0503cb
При оформлении GET и POST запросов необходимо:
1. Указать адрес метода, который вы хотите использовать. Полный список адресов доступен по этой ссылке: http://api.adminsolution.org/Help
2. Параметры необходимые для этого метода. Описание для каждого метода можно найти по этому адресу: http://api.adminsolution.org/Help
3. Авторизационный ключ и тип данных указываются в заголовке.
[Authorization compas:0580fa88-e889-4f88-9e98-fed67e0503cb; Content-Type application/json]
