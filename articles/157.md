# 10.10.2 Расчёт зарплаты. Начало работы

Публичная ссылка: https://app.manajet.org/FAQ/Public/Index/157?culture=ru-RU

Чтобы войти в раздел расчёта безокладной зарплаты, нужно сделать клик на иконке финансового планирования в верхней части программы и затем на вкладке “Расчёт зарплаты”:
Вкладка “Расчёт зарплаты” находится после вкладки “Директива”, поскольку расчёт зарплаты за текущую неделю ФП выполняется после закрытия директивы ФП (после одобрения всех заявок). В результате ФП должна быть известна общая сумма заработной платы, подлежащая распределению по баллам среди сотрудников. Эта сумма должна быть оформлена в виде заявки и одобрена на ФП. Поэтому последовательность именно такая: директива, утверждение заявки с общей суммой зарплаты, закрытие периода ФП, затем распределение “зарплатных” денег между сотрудниками во вкладке “Расчёт зарплаты”.
Права для работы с зарплатой
Для работы со вкладкой “Расчёт зарплаты” пользователю должны быть назначены права.
При включенном параметре “Просмотр в разделе финансового планирования” пользователю доступна вкладка “Расчёт зарплаты” в ФП. При выключенном параметре эта вкладка пользователю недоступна.
Параметр “Редактирование” даёт возможность менять данные в разделе “Расчёт зарплаты”. При выключенном параметры эти данные доступны только в режиме просмотра, без возможности изменения.
“Отчёт” даёт доступ к объединённой зарплатной ведомости, включающей в себя данные нескольких недельных периодов (например, за месяц).
Первоначальные настройки
Первым шагом необходимо добавить настройки расчёта зарплаты в данном периоде ФП. Для этого нужно сделать клик на кнопке “Добавить”:
Открывается форма ввода параметров:
Заявка. В данном периоде ФП необходимо выбрать заявку, которая содержит сумму денег для распределения при расчёте безокладной зарплаты.
Статистика. Нужно выбрать статистику дохода (или маржи). Её график будет показываться в левой части формы, рядом с параметрами.
Баллы за стаж. Этот параметр задаёт, сколько баллов будет добавляться сотруднику за каждый полный год работы. (Дата начала работы сотрудника должна быть для этого внесена в его персональную карточку. См. статью “Добавление сотрудника” в “Справке”.)
Сотрудник вводит состояние статистики. Если данный параметр включён, то каждый сотрудник самостоятельно вводит состояния статистик своих должностей – в своём личном кабинете. Если параметр выключен, то вводить состояния статистик всех сотрудников необходимо тому, кто выполняет расчёт зарплаты – в этом разделе.
Сотрудник вводит коэфф. присутствия. Аналогично предыдущему пункту. Если параметр включён, то каждый сотрудник вводит этот коэффициент самостоятельно, во всплывающем окне своей соответствующей статистики. Если параметр выключен, то коэффициенты присутствия всем сотрудникам нужно вводить “централизованно”, в разделе расчёта зарплаты.
Открыть в списке по умолчанию. Если отметить галочкой этот пункт, то при открытии расчёта з/п Вам будет представлена сразу же данная ведомость. Если же данной отметки не будет, тогда выборка будет представлена из всех существующих ведомостей.
Данные параметры настроек можно менять в каждом периоде ФП. Но, как правило, принятая система настроек один раз используется на протяжении длительного периода времени в компании (при этом изменение параметров в одном периоде ФП не влияет на расчёт зарплаты в других периодах).
