# 5.06.1 Ассортимент. Оборотная ведомость

Публичная ссылка: https://app.manajet.org/FAQ/Public/Index/358?culture=ru-RU

Оборотная ведомость позволяет увидеть всю динамику движения товаров на складе за определенный период времени и проанализировать приходы и расходы связанные с тем или иным ассортиментом. Давайте подробнее рассмотрим функционал оборотной ведомости в программе.
Чтобы увидеть оборотную ведомость, необходимо нажать на значок шестерёнки  в разделе CRM “Ассортимент” и из выпадающего списка выбрать “Оборотная ведомость”:

Папки
Папки позволяют рассортировать между собой карточки ассортимента.
Подробнее о папках в ассортименте можете прочитать в статье, в Справке:https://app.manajet.org/FAQ/Public/Index/18?culture=ru-RU

В расширенном поиске представлены следующие фильтры для поиска карточек ассортимента:
Поиск
Окно поиска позволяет находить карточку ассортимента, если вбить в него её содержимое (ключевые слова, последовательность цифр и т.д.).
Количество
Фильтр для поиска карточек ассортимента по количеству ассортимента.
Счета поставщиков
Данный фильтр позволяет производить отбор ассортимента по счетам поставщиков. Таким образом, если в счёте (или счетах) были указаны категории ассортимента, с помощью данного фильтра можно найти необходимые карточки с товаром.
Дата
Позволяет искать карточки ассортимента по дате операции, проводимой с ассортиментом
Счета клиентам
Данный фильтр позволяет производить отбор ассортимента по счетам клиентам. Таким образом, если в счёте (или счетах) были указаны категории ассортимента, с помощью данного фильтра можно найти необходимые карточки с товаром.
Количество(Тип)
Фильтр с выпадающим списком, позволяющий выбрать определенный тип количества товара (например, карточки с количеством выше нуля, меньше нуля или кроме нуля).
Движение товара
Фильтр с выпадающим списком, позволяющий выбрать определенный тип движения товара (например, карточки с приходом, расходом и др.).
На верхней панели оборотной ведомости расположены следующие наименования: ID, Номер, Название, Начало периода, Приход, Расход, Конец периода, Прибыль, Количество, Сумма в закупочных ценах и Сумма в ценах продажи. Каждое наименование соответствует определенным показателям товара, которые можно отследить и проанализировать в рамках заданного периода.
Накладные внутреннего перемещения
Панель "Накладные внутреннего перемещения" позволяет видеть перемещение товаров между складами.

Импорт: ассортимент
Даёт возможность импортировать карточки ассортимента в программу.

Позволяет экспортировать списки ассортимента в формате Exel.

Даёт возможность просматривать списки импорта и экспорта ассортимента.
Папки
При нажатии на “Папки” откроется окошко добавления новой папки с возможностью редактирования уже созданных папок в программе.
Склад
При нажатии на “Склад” откроется окошко добавления нового склада с возможностью редактирования уже созданных складов в программе.
