# 24.28 Следите за работой в компании с помощью статистик на основе карточек Потока

Публичная ссылка: https://app.manajet.org/FAQ/Public/Index/340?culture=ru-RU

Следить за работой компании стало ещё удобнее - в ManaJet реализованы автоматические статистики на основе карточек “Потока”.
Теперь вы можете создавать отдельные статистики по каждой доске Потока, отбирать данные для статистики по дополнительным колонкам карточек (колонкам, которые содержат целые или десятичные числа – по которым можно рассчитать статистики). Кроме того, можно учитывать статус завершённости карточки.
Таким образом, вы сможете отслеживать насколько эффективно ведётся работа с задачами, а в случае, если вам нужны отдельные статистики по работе каждого работника или по работе с каким-либо проектом — можно воспользоваться отбором данных и указывать сотрудника (или нескольких сотрудников) и глобальные метки, которые отмечены в карточках:
