# 10.08.3 Контроль средств. Настройка способа оплаты

Публичная ссылка: https://app.manajet.org/FAQ/Public/Index/347?culture=ru-RU

Для удобства использования этого раздела и уменьшения вероятности ошибок при работе с Контролем средств можно настроить способы оплаты, которые автоматически связываются с определенными счетами денежных средств.
Это можно сделать нажатием на шестерёнку  внизу, в операциях денежных средств. Из выпадающего списка нужно выбрать “Способы оплаты”:
В способах оплаты нужно нажать на зелёную кнопку “Добавить”:
Например, новым способом оплаты будет "Перечисление", этот метод оплаты может быть связан с определенным банковским счетом, а программа будет предлагать связать этот способ оплаты с определенным счётом денежных средств в разделе "Контроль средств". При оплате счетов, если в счете выбран способ оплаты "Перечисление", программа автоматически свяжет его с определенным счётом денежных средств.
Чтобы сделать это, в открывшемся окошке необходимо написать полное и краткое названия нового способа оплаты и из выпадающего списка выбрать нужную счёт денежных средств:
Такое упрощение использования раздела "Контроль средств" позволит существенно сэкономить время и уменьшить вероятность ошибок при работе с ним.
