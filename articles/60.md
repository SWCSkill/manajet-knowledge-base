# 16.6 Лицензия

Публичная ссылка: https://app.manajet.org/FAQ/Public/Index/60?culture=ru-RU

Лицензия показывает срок действия Вашей программы, количество дней которых осталось до окончания действия, и также количество онлайн-пользователей.
Здесь можно оплатить с помощью банковской карты как продление вашей текущей лицензии  , так и приобрести дополнительные онлайн места, в случае, если вам не хватает онлайн мест для комфортной работы в программе  .
Доступ в эту вкладку есть только у сотрудника с правами “владельца”.
При вводе реквизитов убедитесь в правильности введенной информации. Покупка будет производиться в долларах США. День начала лицензии будет автоматически выставлен днем окончания предыдущей лицензии, либо текущим днем, если со дня окончания уже прошло больше одного дня.
При покупке дополнительных онлайн мест будет рассчитываться сумма с текущеей даты по день окончания действия лицензии.
