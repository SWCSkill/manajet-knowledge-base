# 4.4 Статистики дохода

Публичная ссылка: https://app.manajet.org/FAQ/Public/Index/138?culture=ru-RU

Статистики дохода это статистики трёх типов – выручки, маржинального и скорректированного дохода.
В отличие от обычных статистик, статистики дохода формируются автоматически, по данным блока Финансового Планирования.
Рассмотрим их подробнее и посмотрим, как их можно создать.

Как создать статистики дохода
Добавление статистик в ManaJet выполняется с помощью процедуры из нескольких шагов, где на каждом шаге нужно выбрать или настроить определённые свойства создаваемой статистики. Такой способ настройки в компьютерных программах называется Мастером (в англоязычных программах – “wizard”, что означает “волшебник”.). Способ удобен тем, что на каждом шаге число настроек небольшое и поэтому каждый шаг легко понять.
Для добавления статистики нужно войти в Личный кабинет (иконка ),
выбрать вкладку “Статистики” и нажать кнопку “Добавить статистику”:
Заполняем первую страницу Мастера добавления статистики:
Как и для остальных типов статистик, для статистики дохода нужно назначить ответственного. Выбираем сотрудника из списка. Затем нажимаем кнопку “Далее”.
На второй странице Мастера нужно выбрать один из трёх типов статистики – выручка, маржинальный доход, либо скорректированный доход, :
Период ввода оставляем выбранным по умолчанию – дни.
Период просмотра также не меняем – недели.
Затем нажимаем кнопку “Далее”.

Теперь значения этой статистики будут автоматически обновляться при всех изменениях дохода в Финансовом Планировании – при добавлении или удалении операций дохода.
Аналогичным образом, при помощи Мастера, можно создать статистики маржинального и скорректированного дохода.
Примечание 1.Статистики дохода не обязательно должны быть в единственном экземпляре. Можно создать любое число статистик дохода – для любого числа сотрудников, которым необходимы эти данные.
