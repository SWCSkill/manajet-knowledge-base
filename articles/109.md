# 13.3 Добавление шаблона бизнес-процесса

Публичная ссылка: https://app.manajet.org/FAQ/Public/Index/109?culture=ru-RU

Теперь переходим к самому важному и главному - созданию шаблона бизнес-процесса! На вкладке “Шаблоны бизнес-процессов” нажимаем на кнопку .
У нас откроется окно с добавлением шаблона:
Там где название, указываем название шаблона бизнес-процесса.
Там где цель, нужно написать для чего этот бизнес-процесс будет использоваться. У самого слова Цель есть несколько определений, поэтому их будет выделено несколько:
Решение проблем, которые обычно доставляли человеку терминалы.
Исчерпывающий концепт идеальной картины для любой деятельности. Известный объект (цель), в направлении которого направлено действие с задачей достичь эту цель.
Шаги урегулирования (оценивания), находящиеся в программной форме. Они пронумерованы 1-2-3 и т. д. Или А-Б-В и т. д. Они могут быть расположены в последовательности, в которой они должны быть сделаны, но более важно, когда один человек или одна команда собирается сделать все дело целиком, шаг за шагом. Эти шаги называются целями. Каждая часть программы (каждая цель) назначается кому-либо для исполнения или выполнения.
Стремление, которое человек намеревается выполнить в течение установленного периода времени.
Далее, идет поле “Описание”. Где Вы можете описать данный шаблон бизнес-процесса.

Для того чтобы добавить шаг нажмем на кнопку .
Название - пишем название самого шага.
Подразделения - указываем пост, который будет выполнять данные шаг.
Также шаг может выполнять полностью все подразделение. Например, сектор или секция, тогда выбираем “Не выбран”.
Также ответственным за данный шаг Вы можете указать такого же сотрудника, который ответственный за любой из предыдущих шагов, нажав на кнопку “Шаг”.
И посмотрим оставшиеся поля:
Описание - описание данного шага(информация) или более развернутое описание шага, который должен быть выполнен;
Срок исполнения - указываем количество и рядом из выпадающего списка выбираем нужный период:
Если данный шаг является первым, то есть тем которым будет начинать данную бизнес-процесс, то ставим галочку, если данный шаг будет завершать БП, то тоже - не забываем ставить галочку!
Если Вы хотите выводить данные по этому шагу данного бизнес-процесса на плазму, то тоже ставим галочку. Более подробно о том, как выводить шаги бизнес-процесса на плазму рассказано в разделе "Плазма".
Сохраняем данные и переключаемся на следующая вкладку "Подшаги".
Здесь будут создавать подшаги(Если наш шаг очень большой и подразумевает выполнения нескольких заданий). Если нам не нужно разбивать шаг на несколько действий, то тогда ничего здесь не заполняем. Отметка "Условный" говорит о том, является ли данный подшаг обязательный к выполнению или нет. Если нет, то обязательно поставить галочку.
Также в шаге Вы можете создать дополнительные поля(для указания какой то информации). Если дополнительное поле обязательно для заполнения в данном подшаге, то ставим галочку "Обязательно для заполнения". Если надо чтобы данный подшаг был отображен в расширенном поиске, для отбора бизнес-процесса по данному параметру в будущем, то ставим галочку. Если необходимо то в описании Вы можете описать для чего данный подшаг нужен.
Переходим на следующую вкладку - "Переходы".
Входящий переход - здесь выбирается предыдущий шаг, после завершения которого мы перейдем в текущий. Его нужно выбрать из выпадающего списка.
Исходящий переход - тот переход, который выбирается следующий шаг, в который мы перейдем после завершения текущего. Его тоже нужно выбрать из выпадающего списка.
Переходы нужно указывать обязательно. Поскольку шаги в бизнес-процессе должны выполняться последовательно.
И последняя вкладка которая у нас здесь есть - "Должностные инструкции". К шагу вы можете прикрепить инструкцию - документы, которые уже должны быть у Вас в программе в разделе “Должностные инструкции”. Чтобы прикрепить ссылку на документ к данной бизнес-процессе, то ее необходимо будет указать галочкой из списка. Для более расширенного поиска документа можно воспользоваться строкой для поиска.
После этого нужно обязательно нажать на кнопку “Сохранить”, для сохранения всех внесенных данных.
