Настройки
В разделе настроек счетов поставщиков и клиентов предоставляется возможность тонко настраивать параметры, связанные с обработкой счетов и взаиморасчетами между компанией и ее контрагентами. Ниже перечислены основные пункты, доступные для настройки в этом разделе:

Автоматический номер на основе серии
Данная функция упрощающает работу с серийными номерами в новых счетах.
Автоматическое заполнение серийного номера: При указании серии счёта, программа автоматически подставляет его номер. Это работает только для новых счётов.
Для числовых номеров: Если введенный номер является числом, программа автоматически увеличивает его на единицу относительно предыдущего номера счёта. Например, если последний номер счёта был 100, новый счёт получит номер 101.
Для нечисловых номеров: Если номер не является числом, программа использует номер последнего счёта, предоставляя пользователю возможность самостоятельно его исправить. Это дает гибкость в учете счётов, которые не следуют стандартной числовой нумерации.
Для новых серий: В случае если счёта с введенной серией еще не существовали, программа автоматически присваивает номер "1". Это упрощает начало работы с новыми сериями счётов, избавляя от необходимости вручную указывать начальный номер.
Это обновление упрощает управление нумерацией счётов, автоматизируя рутинные процессы и минимизируя вероятность ошибок при вводе данных.
Настройки уникальности (Счета поставщика / Счета клиенту)
Вы можете настроить уникальность номеров счетов, чтобы избежать их повторения. Также можно установить период для поддержания уникальности, например, год или месяц, и задать уникальные номера для каждой отдельной компании.
Карточки ассортимента
Вы можете настроить уникальность серийных номеров в карточках ассортимента, чтобы избежать их дублирования. Кроме того, можно утвердить или запретить возможность наличия минусовых остатков товара.
НДС (Налог на добавленную стоимость)
Данный раздел позволяет задать правила для обработки НДС на счетах как поставщиков, так и клиентов. Для счетов клиентам можно выбрать способ подсчета НДС:
Розничный: НДС включен в цену товара.
Оптовый - НДС не включён в цену товара.
Аналогично, для счетов клиентам также предоставляется выбор способа подсчета НДС:
Розничный: НДС включен в цену товара.
Оптовый - НДС не включён в цену товара.

Здесь можно настроить уникальность счетов, чтобы избежать дублирования номеров счетов среди разных контрагентов.

В шаблоне указаны названия используемых полей для создания счета, они написаны на английском, чтобы при импорте программа могла их распознать соответствующим образом. Если эта галка не стоит, то счета будут идти сразу с первой строки, а наименования полей будут выставлены по умолчанию.
