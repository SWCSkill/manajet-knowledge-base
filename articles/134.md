# 3.2 Добавление сотрудника

Публичная ссылка: https://app.manajet.org/FAQ/Public/Index/134?culture=ru-RU

Для добавления сотрудника нужно войти в список персонала – кликом на иконке  в верхней части страницы.
Затем кликнуть на кнопке  (кнопка доступна только пользователю с назначенным правом на добавление сотрудника. Ссылка на статью о назначении прав доступа – внизу этой статьи).
Откроется форма для ввода новой учётной записи сотрудника:
Эти поля необходимо заполнить.
Общие данные
Имя — графа для ввода имени.
Фамилия — графа для ввода фамилии.
Отчество — графа для ввода отчества.
Должность — графа для ввода должности.
В группе — можно добавить пользователя в группу, объединённую по какому-либо принципу. К примеру:

Примечание:
Больше информации о группировке сотрудников Вы можете получить в этой статье.
Пол — в данной графе можно выбрать пол сотрудника.
День рождения — в этой графе можно указать день рождения сотрудника.
Начало работы — в данной графе можно указать начало работы в компании данного сотрудника.
Окончание работы — в данной графе можно указать дату окончания работы в компании данного сотрудника.
Дополнительные колонки
С помощью дополнительных колонок можно внести дополнительную информацию по сотруднику в карточку контакта.
К примеру, можно внести дополнительный телефон, email или же отдельную колонку для образования и Skype:
Контактные данные и вход
E-mail — графа для ввода email-а.
Телефон — графа для ввода контактного телефонного номера.
Логин для входа — логин, с помощью которого пользователь сможет войти в программу.
Пароль для входа — пароль, с помощью которого пользователь сможет войти в программу.
Если нажать кнопку , программа сохранит введённые вами данные, но форма добавления останется открытой и вы сможете продолжить ввод.
При нажатии на , программа сохранит данные и закроет форму, перейдя снова в список персонала.
Нажатие кнопки  также переведёт вас в список персонала, но без сохранения данных.
