# 5.06.2 Ассортимент. Движение товара

Публичная ссылка: https://app.manajet.org/FAQ/Public/Index/368?culture=ru-RU

Движение товара — это записи о перемещении товаров между складами, а также о продажах, закупках и ценах. Эти записи позволяют отслеживать, какие товары поступают на склад, какие реализуются, а также контролировать уровень запасов.
Чтобы увидеть движение товара, нужно кликнуть на значок шестерёнки   в разделе CRM “Ассортимент” и из выпадающего списка выбрать “Движение товара”:

Панель “Движение товара”
Карточки ассортимента, с которыми проводились операции получения товара и отправки, расположены в виде списка:
Номер
Этот столбец просто отображает порядковый номер каждой записи в таблице.
Название
Здесь указывается название товара, к которому относится данная запись.
Получение товара
В этом разделе указываются данные о полученных товарах:
Количество: Сколько единиц товара было получено.
Цена: Цена за одну единицу товара.
Всего: Общая стоимость полученных товаров (количество умноженное на цену).
Безналичные: Сумма, оплаченная безналичными средствами.
Наличные: Сумма, оплаченная наличными.
Отправленные товары
Здесь указываются данные о товарах, которые были отправлены:
Количество: Сколько единиц товара было отправлено.
Закупочная цена: Цена, по которой товар был куплен.
Цена продажи: Цена, по которой товар был продан.
Сумма в закупочных ценах: Общая стоимость отправленных товаров по закупочным ценам (количество умноженное на закупочную цену).
Сумма в ценах продажи: Общая стоимость отправленных товаров по ценам продажи (количество умноженное на цену продажи).
Наличные: Сумма, полученная наличными за отправленные товары.
Безналичные: Сумма, полученная безналичными средствами за отправленные товары.

В расширенном поиске представлены следующие фильтры для поиска карточек ассортимента:
Поиск
Окно поиска позволяет находить карточку ассортимента, по её содержимому (ключевые слова, последовательность цифр и т.д.).
Количество
Фильтр для поиска карточек ассортимента по количеству ассортимента.
Счета поставщиков
Данный фильтр позволяет производить отбор ассортимента по счетам поставщиков. Таким образом, если в счёте (или счетах) указаны категории ассортимента, с помощью данного фильтра можно найти необходимые карточки с товаром.
Дата
Позволяет искать карточки ассортимента по дате операции, проводимой с ассортиментом
Счета клиентам
Данный фильтр производит отбор ассортимента по счетам клиентам. Таким образом, если в счёте (или счетах) указаны категории ассортимента, с помощью данного фильтра можно найти необходимые карточки с товаром.
Количество (Тип)
Фильтр с выпадающим списком, позволяющий выбрать определенный тип количества товара (например, карточки с количеством выше нуля, меньше нуля или кроме нуля).
Движение товара
Фильтр с выпадающим списком, позволяющий выбрать определенный тип движения товара (например, только карточки с приходом или расходом).
На верхней панели оборотной ведомости расположены следующие наименования: номер, название,  получение товара (количество, цена, всего, наличные), отправленные товары (количество, закупочная цена, цена продажи, сумма в закупочных ценах, сумма в ценах продажи, безналичные, наличные). Каждое наименование соответствует определенным показателям товара, которые можно отследить и проанализировать в рамках заданного периода.
