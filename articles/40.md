# 10.03.1 Фонды

Публичная ссылка: https://app.manajet.org/FAQ/Public/Index/40?culture=ru-RU

Одно из определений слова фонд – это «ресурс, запас чего-либо». В ManaJet фонд – это запас денежных средств, отложенных для какой-то определенной цели: например, для рекламы и продвижения, для закупок товара, как резерв и т.д. Фонды в технологии Финансового Планирования используются очень активно.
Фонды можно создавать и настраивать во вкладке “Фонды”. Чтобы войти в эту вкладку, нужно кликнуть на ее название, либо кликнуть сначала на иконку , потом перейти во вкладку “Фонды”.
Во вкладке находятся две панели. В верхней находятся сами фонды, с рассчитанными итогами по ним в этом периоде ФП. В нижней – операции с фондами.

Номер фонда
Нужно иметь в виду, что номера любых финансовых категорий не должны повторяться. Номер вида фонда не должен совпадать ни с номером любого другого вида фонда, ни с номером дохода либо статьи расхода. Поэтому номера фондов лучше начинать с определенной буквы, например F.
Обычно используется такое сочетание “F” (фонд) и “01”.

Тип

Тип фонда
Мы можем выбрать между рабочим и накопительным фондом. Фонды с типом “рабочий” (этот тип выбран по умолчанию) можно использовать для одобрения заявок и счетов к оплате. Фонд с типом “накопительный” не будет появляться в заявках и счетах. Он предназначен, как уже понятно по названию, только для накопления средств.
Этап распределения
Один из трех: выручка, маржинальный доход и скорректированный доход.
Выручка – все средства, поступившие в компанию за период финансового планирования.
Маржинальный доход = выручка минус отчисления в фонды выручки.
Скорректированный доход = маржинальный доход минус отчисления в фонды маржинального дохода. (Также можно создать фонды, куда откладываются средства со скорректированного дохода).
Типы дохода можно также назвать этапами начисления средств по фондам.
Типы дохода не следует путать с видами дохода.
Описание
Текстовая информация о фонде, примечание.
Цвет
Можно выбрать цветовое цвет фонда.
Перемещать средства
Если установлена эта отметка, то фонд может участвовать в операции перемещения средств из одного фонда в другой.
Фонд ФД
Отметка “Фонд ФД” ставится для фондов Финансового Директора. Как вы видите в подсказке внизу, это особые фонды, доступные только Финдиректору, собственнику и пользователям с особыми правами. То есть другим пользователям, даже имеющим доступ к разделу ФП, к вкладкам “Фонды” и “Директива”, эти фонды видны не будут. Такие фонды можно использовать для хранения средств учредителей, средств строительного фонда или других особо важных резервов.

Архивирование фонда
Фонды не могут быть удалены, а только перенесены в архив. Из архива записи могут быть вновь восстановлены, т.е. перенесены в активные.
Фонды можно перенести в архив либо по одному, либо сразу несколько.
