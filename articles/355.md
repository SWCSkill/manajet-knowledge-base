# 14.4 Текстовые команды в ManaJet

Публичная ссылка: https://app.manajet.org/FAQ/Public/Index/355?culture=ru-RU

Текстовые команды — это инструкции, позволяющие автоматически подставлять в текст генерируемого в ManaJet документа (договора, счёта клиенту, акта выполненных работ, письма для рассылки и любых других) нужное данное из программы – название компании, адрес, имя человека и прочие.
Так в ManaJet можно создавать разные шаблоны для генерации документов. Это удобно, например, для генерации бухгалтерских документов. Также это дает возможность распечатывать один и тот же документ на двух разных языках - с помощью двух заранее подготовленных шаблонов. Или, если в одних случаях, нужно формировать документ с НДС, а в других – без него. Это также решается при помощи шаблонов и текстовых команд в них.
В письмо рассылки с помощью текстовой команды можно автоматически подставлять имя адресата. И тогда это письмо – уже не безликий спам, а почти личное послание конкретному человеку.

Тогда, при отправке рассылки, получатель увидит письмо следующим образом:
Подробнее о текстовых командах:
{{{company.name}}}
Подставляет название компании или полное имя (имя и фамилию) частного лица в карточке CRM.
{{{company.first_name}}}
Если адресат – частное лицо, то будет отображаться имя. Если адресат компания — ничего не будет отображаться.
{{{company.middle_name}}}
Для адресата частного лица будет отображаться отчество. Если адресат компания — ничего не будет отображаться.
{{{company.last_name}}}
Фамилия адресата – частного лица. Если адресат компания — ничего не будет отображаться.
{{{company.is_private_person}}}
Возвращает true (“истинно”), если адресат является частным лицом. Используется с условными командами.
{{{company.is_customer}}}
Создаётся условие, чтобы отображались данные поставщика (т.е. если в карточке CRM компания является клиентом — ничего не будет отображаться).
{{{company.is_vendor}}}
Создаётся условие, чтобы отображались данные клиента (т.е. если в карточке CRM компания является поставщиком — ничего не будет отображаться).
{{{company.registration}}}
ИНН (регистрационный номер) компании в CRM.
{{{company.kpp}}}
КПП компании в CRM.
{{{company.birthday}}}
День рождения частного лица в CRM.
{{{company.gender}}}
Если адресат частное лицо, будет обозначен пол частного лица. Если же компания — ничего не будет отображаться. Используется с условными командами.
{{{company.outer_id}}}
ID из внешней программы.
{{{company_contact_info company.company_contact_info "Phone"}}}
Контактный телефон компании.
{{{company_contact_info company.company_contact_info "Email"}}}
Контактный email-адрес компании.
{{#company.is_private_person}}Частное лицо{{/company.is_private_person}}
Если это частное лицо, будет отображаться слово “Частное лицо”.
{{^company.is_private_person}}Компания{{/company.is_private_person}}
Если это компания, будет отображаться слово “Компания”.
{{#company.gender}}Мужской{{/company.gender}}
Создаёт надпись “Мужской”.
{{^company.gender}}Женский{{/company.gender}}
Создаёт надпись “Женский”.
{{{company_contact.name}}}
Если у компании есть контакт, то указывается полное имя этого контакта (ФИО).
{{{company_contact.first_name}}}
Имя контакта из карточки компании в CRM.
{{{company_contact.middle_name}}}
Отчество контакта из карточки компании в CRM.
{{{company_contact.last_name}}}
Фамилия контакта из карточки компании в CRM.
{{{company_contact.gender}}}
Пол контакта из карточки компании в CRM.
{{{company_contact.is_private_person}}}
Команда позволяет увидеть текст, только если контакт является частным лицом. Для этого необходимо слева и справа текста расположить данную команду.
Например: {{#company_contact.is_private_person}}текст{{/company_contact.is_private_person}}
{{{company_contact_info company_contact.company_contact_info "Phone"}}}
Телефон первого контакта из карточки CRM.
{{{company_contact_info company_contact.company_contact_info "Email"}}}
Email-адрес первого контакта из карточки CRM.
{{#company_contact.gender}}Мужской{{/company_contact.gender}}
Текст внутри команды будет в мужском роде, само слово “мужской” можно заменить на любое другое слово. В случае,  если контакт из карточки CRM женского пола, он это слово не увидит.
{{^company_contact.gender}}Женский{{/company_contact.gender}}
Текст внутри команды будет в женском роде, само слово “женский” можно заменить на любое другое слово. В случае,  если контакт из карточки CRM женского пола, он это слово не увидит.
%Отказаться от рассылки%
Ссылка в письме, дающая пользователю возможность отказаться от рассылки. Текст ссылки будет сформирован на том языке, который выбран в рассылке.
