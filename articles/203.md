# 12.06 Автоматизация действий в Потоке

Публичная ссылка: https://app.manajet.org/FAQ/Public/Index/203?culture=ru-RU

Некоторые действия в “Потоке” можно автоматизировать. За это отвечает пункт “Авто-действия”. Одно из таких действий –  автоматическое назначение ответственного за карточку.  Так можно автоматически назначить одного из менеджеров по продажам, когда заявка приходит с сайта. Или бухгалтера –  когда карточка попадает на этап оформления документов (на одну из колонок доски).

Рассмотрим все возможности по отдельности:
Добавить ответственного в карточку
Позволяет автоматически назначить того или иного сотрудника ответственным во всех карточках выбранного Вами списка.
Удалить ответственного с карточки
Позволяет автоматически удалить всех назначенных ответственных при том или ином условии в карточках выбранного Вами списка.
Удалить всех ответственных с карточки
Данная кнопка удаляет всех ответственных с карточки.
Добавить наблюдателя
Позволяет автоматически добавить наблюдателя (к примеру, руководителя, для возможности контроля процессов в списке).
Чтобы добавить одно (или несколько) из перечисленных выше условий нужно выбрать ответственного и нажать на  или же .
Отметить карточку как завершенную
Позволяет автоматически добавить карточке, перенесённой в выбранный список, статус “Завершенная”.
Отметить карточку как незавершенную
Позволяет автоматически добавить карточке, перенесённой в выбранный список, статус “Незавершенная”.
