# 12.05 ManaJet Поток: Дополнительные поля

Публичная ссылка: https://app.manajet.org/FAQ/Public/Index/486?culture=ru-RU

Для удобства, в карточку компании можно добавлять поля. Для этого на странице Потока нужно нажать на шестерёнку и выбрать “Дополнительные поля”:
В появившемся окне следует нажать на кнопку :
Откроется карточка рабочего процесса:
В ней следует указать:
Тип колонки
Здесь можно выбрать формат добавляемой колонки.
Адрес - добавляет дополнительную строку для заполнения адреса.
Воронка продаж - добавляет в карточку уже созданную воронку продаж.
Да/Нет - добавляет дополнительную галочку в форме редактирования объекта.
Дата - добавляет дополнительную строку для заполнения даты.
Десятичное число - добавление числа с дробным значением.
Компания - позволяет привязать уже созданную компанию.
Контрольный список - добавляет возможность создавать и привязывать списки.
Персонал - возможность указания сотрудника.
Список - создает выпадающий список.
Ссылка - позволяет привязать ссылку.
Счет клиенту/Счет поставщика - привязывает уже имеющийся в программе счет к объекту.
Текст - возможность ввода однострочного текста с данными.
Текст, многострочный - возможность ввода многострочного текста с данными.
Файл - возможность прикреплять файл (можно с помощью набора клавиш Ctrl+v).
Целое число - возможность ввода целого числа.
Название
Это то, как поле будет отображаться в карточке, её наименование в ней.
Описание
Подробное описание данной колонки.
Показывать в списке
Отмечая этот пункт, Вы выводите данное поле и заполненную в ней информацию в основной список объектов для того, чтобы ее было видно без раскрытия полной карточки.
Обязательно для заполнения
При отметке этого пункта рядом с созданной колонкой появится значок . Он означает, что без заполнения этого поля программа не даст сохранить запись – ни при создании, ни при редактировании.
Показывать на основной вкладке
Эта отметка относится только к созданию дополнительных колонок в разделе “Компании”, в CRM. Если она включена, то поле будет показано во вкладке “Основные” панели с информацией о компании. Если она не отмечена, то дополнительные колонки отображаются во вкладке “Дополнительно”.
