# 16.3 Источники данных

Публичная ссылка: https://app.manajet.org/FAQ/Public/Index/62?culture=ru-RU

“Внешний запрос” для статистик, этот инструмент поможет загрузить значения статистики в программу. Для его использования потребуется строка подключения (англ. Connection string) к источнику и текст запроса для получения списка значений. Для примера будем использовать для MS SQL.
Строка подключения к базе данных - это строка, которая содержит информацию, необходимую ODBC драйверу/OLE DB провайдеру для подключения к источнику данных. В качестве источника данных могут выступать, например, базы данных, файлы с данными)
Provider=SQLOLEDB;Data Source=IP;Initial Catalog=my_home;user id=User; password=*******
В ней необходимо указать IP адрес, порт подключения, название базы данных, имя пользователя и пароль к базе данных.
Текст запроса лучше подготовить заранее, воспользовавшись сторонними аппликациями. Если запрос будет составлен неверно, то программа сообщит только о получении пустого результата. Пример текста запроса:
SELECT
dt_short AS period_begin,
dt_short AS period_end,
count(1) AS real_value,
0 AS is_quota
FROM [dbo].[error_handler]
WHERE url NOT LIKE '%DashBoard%'
AND application NOT LIKE 'localization'
GROUP BY dt_short
ORDER BY dt_short DESC
ВАЖНО, чтобы столбцы были подписаны (period_begin, period_end, real_value, is_quota), где
period_begin - начало периода,
period_end - конец периода,
real_value - значение статистики,
is_quota - квота (0 - нет/ 1 - да).
После заполнения всей информации для “Внешнего запроса” необходимо убедиться, что подключение возможно и текст запроса написан верно, для этого нужно нажать кнопки “Тест соединения” и “Тест запроса”. Если соединение не удалось или запрос написан неверно, то программа всё равно позволит сохранить настройки.
Планировщик. В этом разделе необходимо указать, с какой периодичностью будут запрашиваться обновление данных. Если в программе присутствует несколько “Внешних запросов”, рекомендуем ставить разное время запуска, чтобы не перегружать ваш сервер.
При создании статистики указать тип статистики - “Внешняя”, и выбрать из какого “Внешнего запроса” получать данные.
