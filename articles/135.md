# 3.4 Назначение прав доступа

Публичная ссылка: https://app.manajet.org/FAQ/Public/Index/135?culture=ru-RU

При открытии этой вкладки, по умолчанию открыт раздел прав, относящийся к финансовому планированию. Сделаем клик на панельке с надписью “Финансовое планирование”:
Раздел “Финансовое планирование” закроется:

Права доступа расположены четырьмя блоками:

Каждый из блоков прав доступа можно открыть и закрыть кликом на его заголовке:
Рассмотрим блок прав подробнее на примере “Финансового планирования”.
Права внутри блока расположены секциями:
Каждая секция относится к работе в одной из вкладок ФП:
Внутри секции одни права зависят от других. В таких случаях они расположены “лесенкой”,  по принципу подчинённости:
Так в секции “Денежные средства” право “Редактирование” зависит от права “Просмотр”. Если пользователь не имеет права входить во вкладку “Денежные средства”, то ему тем более должно быть недоступно редактирование там данных.
Поэтому пока пользователю не назначено право “Просмотр”, право “Редактирование” недоступно для назначения.
Важно!В компании может быть только один Финансовый директор. Поэтому право “Финансовый директор” может быть назначено только одному сотруднику в компании. Если такое право уже назначено одному сотруднику и вы назначаете другому, то у первого сотрудника оно автоматически снимается.
