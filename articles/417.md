# 24.55 Новая функция в ManaJet: автоматизация серийных номеров счетов!

Публичная ссылка: https://app.manajet.org/FAQ/Public/Index/417?culture=ru-RU

В программе ManaJet добавлена новая функция, упрощающая работу с серийными номерами в новых счетах:
🔍 Автоматическое заполнение серийного номера: При указании серии счёта, программа автоматически подставляет его номер. Это работает только для новых счётов.
🔢 Для числовых номеров: Если введенный номер является числом, программа автоматически увеличивает его на единицу относительно предыдущего номера счёта. Например, если последний номер счёта был 100, новый счёт получит номер 101.
✏️ Для нечисловых номеров: Если номер не является числом, программа использует номер последнего счёта, предоставляя пользователю возможность самостоятельно его исправить. Это дает гибкость в учете счётов, которые не следуют стандартной числовой нумерации.
🆕 Для новых серий: В случае если счёта с введенной серией еще не существовали, программа автоматически присваивает номер "1". Это упрощает начало работы с новыми сериями счётов, избавляя от необходимости вручную указывать начальный номер.
Это обновление упрощает управление нумерацией счётов, автоматизируя рутинные процессы и минимизируя вероятность ошибок при вводе данных.
