# 10.04.4 Расходы. Периоды ФП 1

Публичная ссылка: https://app.manajet.org/FAQ/Public/Index/44?culture=ru-RU

Кнопка  открывает инструмент “Периоды ФП 1”. Он отделен от обычного недельного планирования, позволяет планировать на большие периоды (по умолчанию на месяц). Доступна для должности с правами Финансовый Директор.
Чтобы создать период, нужно нажать кнопку .
Откроется форма для добавления, где нужно указать:
Название - название создаваемого периода, задается произвольно.
Период - нужно выставить дату начала и дату окончания. По умолчанию выставлен месяц. Даты одного периода не должны пересекаться с датами другого.
Доходы - здесь доступно заполнение суммы, которую получить по каждому виду дохода за этот период. Колонка будет заполнена автоматически, когда наступят стандартные недельные периоды ФП, и в них будут внесены данные по доходам. Колонка показывает разницу между запланированной и реальной суммой.
Расходы - здесь доступно заполнение суммы, которую получить по каждой статье расхода за этот период. Колонка будет заполнена автоматически, когда наступят стандартные недельные периоды ФП, и в них будут внесены данные по расходам. Колонка показывает разницу между запланированной и реальной суммой.
Кнопка сохраняет внесенные данные, оставаясь в этом же окне.
Кнопка сохраняет данные и полностью закрывает окно периодов ФП1.
Кнопка возвращает в общий список периодов ФП1 без сохранения.
Кнопка  закрывает окно периодов ФП1 без сохранения.
Если нужно отредактировать период ФП1, то нужно нажать на его название в общем списке периодов ФП1.
В общем списке периодов ФП1 по каждому периоду показаны запланированные и реальные суммы по всем видам доходов и по всем расходам. Здесь колонка показывает разницу между реальным доходом и реальным расходом.
Кнопка копирует период. Открывается форма создания, заполненная данными копируемого периода.
Кнопка удаляет период. Архива не существует, поэтому удаляются периоды полностью, без возможности восстановления.
Поиск позволяет отбирать периоды по названию. Кнопка открывает расширенный поиск, где еще доступен отбор по датам. Кнопка  сбрасывает все параметры поиска.
