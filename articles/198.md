# 19.1 Интеграция с Google-календарём

Публичная ссылка: https://app.manajet.org/FAQ/Public/Index/198?culture=ru-RU

В окошке отображается полное наименование задач, указана компания, дата начала срока исполнения задачи и её дедлайн, а также активная ссылка, по которой можно перейти в ManaJet и просмотреть данную задачу уже в программе.
Справа в окошке, вверху, отображается инструментальная панель:
— “Изменить мероприятие” — это кнопка редактирования.
При нажатии на неё, задача откроется полностью с возможностью редактирования.
— “Удалить мероприятие” — кнопка для мгновенного удаления задачи.
— “Отправить данные по эл. почте” — это кнопка предоставит возможность отправить задачу по электронной почте:
— “Параметры” — при нажатии на эту кнопку всплывёт окошко со списком возможностей действий с данной задачей:
печать задачи;
копирование задачи в календаре;
копирование задачи в гугл-аккаунте (для дальнейшего доступа к ней одного аккаунта);
публикация задачи (для открытого доступа к задаче);
смена владельца (для смены ответственного за данную задачу);
