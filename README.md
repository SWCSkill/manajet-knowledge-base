# ManaJet Knowledge Base (для swc-manajet-expert)

Справочный корпус ManaJet, отобранный для использования AI-ассистентом сотрудника SWC.

## Что внутри

- `articles/` - 221 файл `{id}.md`, по одной статье на файл
- `INDEX.md` - индекс всех статей с разбиением по разделам

## Источник

Корпус получен из официальной справки ManaJet (https://app.manajet.org/FAQ/Public/Index/0).
Раздел 24 "Минутка ManaJet" отфильтрован: оставлены только статьи о возможностях программы,
общеделовой блог-контент исключён.

## Как использовать

Скилл `swc-manajet-expert` обращается к этому репо через curl по схеме:

```bash
# Получить индекс
curl -s https://raw.githubusercontent.com/<owner>/<repo>/main/INDEX.md

# Получить конкретную статью по ID
curl -s https://raw.githubusercontent.com/<owner>/<repo>/main/articles/165.md
```

## Обновление

При изменениях в справке ManaJet:
1. Выгрузить актуальный JSON из источника
2. Прогнать скрипт сборки
3. Закоммитить изменения

Скилл подхватит обновления при следующем обращении - пересобирать SKILL.md не нужно.

## Лицензия и атрибуция

Тексты статей принадлежат правообладателям ManaJet. Этот репозиторий - локальная копия
для внутреннего использования сотрудниками SWC при работе с системой.
