Карта корпуса - 221 статья. По этой карте определяй ID нужной статьи.

У каждой статьи две ссылки:
- **Публичный URL** - даётся сотруднику в ответе. Открывается без авторизации, содержит скриншоты интерфейса.
- **curl-URL** - используешь сам для подтяга текста через bash_tool.

**Шаблоны:**
```
Публичный: https://app.manajet.org/FAQ/Public/Index/{ID}?culture=ru-RU
curl:       https://raw.githubusercontent.com/SWCSkill/manajet-knowledge-base/main/articles/{ID}.md
```
