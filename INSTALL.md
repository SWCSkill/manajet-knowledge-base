# Установка swc-manajet-expert

Эта инструкция - для сотрудников SWC, которые хотят установить себе скилл для работы с ManaJet.

## Шаг 1. Скачать актуальный скилл

Постоянная ссылка на последнюю версию:

**https://github.com/SWCSkill/manajet-knowledge-base/releases/latest/download/swc-manajet-expert.skill**

Эта ссылка всегда ведёт на самую свежую версию, никаких ручных проверок не нужно.

## Шаг 2. Загрузить в Claude.ai

1. Открой Claude.ai
2. Зайди в **Settings** → **Capabilities** → **Skills**
3. Нажми **Upload skill**
4. Выбери только что скачанный файл `swc-manajet-expert.skill`
5. Жди подтверждения, что скилл установлен

## Шаг 3. Разрешить доступ к GitHub

Скилл подгружает статьи справки ManaJet через GitHub. Чтобы это работало, нужно один раз добавить домен в разрешённые:

1. В Claude.ai: **Settings** → **Capabilities** → **Code execution**
2. Найди поле **Additional allowed domains**
3. Добавь: `raw.githubusercontent.com`
4. Сохрани

## Шаг 4. Проверить

Открой новый чат в Claude.ai и напиши что-нибудь в духе:

> Как создать формульную статистику в ManaJet?

Скилл должен:
- Понять, что вопрос про ManaJet
- Найти статью 165 «Формульная статистика»
- Дать чёткие шаги
- В конце дать публичную ссылку на статью со скриншотами:
  `https://app.manajet.org/FAQ/Public/Index/165?culture=ru-RU`

Если работает - всё ок. Если нет - см. ниже.

## Если что-то не работает

**Скилл не активируется на вопросах про ManaJet.** Перепроверь, что в Settings → Capabilities → Skills он включён (галочка стоит).

**Скилл говорит, что не может подгрузить статью.** Значит, либо `raw.githubusercontent.com` не добавлен в allowlist (вернись к Шагу 3), либо у тебя корпоративный аккаунт без права менять настройки. В последнем случае скилл переключится в облегчённый режим: будет давать названия статей и публичные ссылки, без полного содержимого.

**Скилл выдаёт устаревшую информацию.** Возможно, у тебя установлена старая версия. Скачай заново по ссылке выше и переустанови.

## Обновление

При выходе новой версии:
1. Скачай заново по постоянной ссылке (она всегда ведёт на последнюю)
2. В Claude.ai: Settings → Skills → удали старую версию `swc-manajet-expert`
3. Загрузи новую через Upload skill

История версий: см. [CHANGELOG.md](./CHANGELOG.md)

## Если хочешь конкретную версию

В разделе [Releases](https://github.com/SWCSkill/manajet-knowledge-base/releases) выбери нужный тег и скачай оттуда.

Прямые ссылки на конкретные версии имеют такой вид:
```
https://github.com/SWCSkill/manajet-knowledge-base/releases/download/v1.3.0/swc-manajet-expert.skill
```
