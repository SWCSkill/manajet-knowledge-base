# Changelog

История версий скилла swc-manajet-expert.

## [1.3.0] - 2026-04-28

### Добавлено
- Шаг 0 в рабочем цикле: проактивная проверка доступа к репо в начале сессии (один curl на INDEX.md). Сотрудник не упирается в ошибку посреди работы.
- Развёрнутый блок «Помощь сотруднику с настройкой доступа» - готовый скрипт ответа с пошаговой инструкцией: куда зайти в Claude.ai, какой домен добавить, как проверить.
- Альтернативный режим работы без curl-доступа: скилл может работать только по локальному INDEX.md, отправляя сотрудника читать статьи по публичной ссылке.

### Изменено
- Раздел диагностики переоформлен в таблицу для случаев после успешной настройки доступа (404, 403 и пр.).

## [1.2.0] - 2026-04-28

### Добавлено
- Обязательное требование давать публичный URL `https://app.manajet.org/FAQ/Public/Index/{ID}?culture=ru-RU` в каждом ответе сотруднику. Эти страницы открываются без авторизации и содержат скриншоты интерфейса.
- В шапке каждого файла статьи в репо проставлена своя публичная ссылка.
- В корневом INDEX.md репо у каждой статьи две ссылки: на локальный `.md` и на публичную справку.

### Изменено
- Все внутренние ссылки `app.manajet.org/FAQ/Home/Index/{id}` заменены на `Public/Index/{id}?culture=ru-RU` (требуют авторизации vs не требуют).

## [1.1.0] - 2026-04-28

### Добавлено
- Файл `references/swc_practice.md`: SWC-надстройка над справкой ManaJet. Включает 12 шагов первичного внедрения и расширенный блок про Направляющие формы (НФ) с переосмыслением через HMS.
- В `swc_terminology.md`: правка про «пост vs должность» (оба термина живые в SWC), добавлены частица, НФ ↔ Бизнес-процесс, ИЦО, Коммуникационный центр.

### Источник дополнений
Материалы коллеги (методичка «Часть 2 - ManaJet как операционная система компании») с фильтрацией дублей справки.

## [1.0.0] - 2026-04-28

### Первый релиз
- SKILL.md с маршрутизацией запросов и архитектурой работы скилла
- `references/INDEX.md`: карта 221 статьи справки ManaJet по 24 разделам
- `references/swc_terminology.md`: словарь соответствий ManaJet ↔ SWC
- Корпус справки в `articles/`: 221 файл по одной статье на ID
- Гибридная архитектура: локальные референсы для навигации + curl на GitHub за статьями
